# 行動規範

このプロジェクトは Contributor Covenant の行動規範に従います。

## 私たちの基準

前向きなコミュニティづくりに寄与する行動の例:
- 相手を尊重したコミュニケーション
- 建設的なフィードバック
- 背景の異なる貢献者への共感

受け入れられない行動の例:
- ハラスメントや差別
- 個人攻撃
- 同意のない個人情報の公開

## 執行

メンテナーは、行動規範に反するコメント・コミット・コード・Issue・その他の投稿を、編集・拒否・削除できるものとします。

## 報告

行動規範違反は、リポジトリメンテナーへ非公開で報告してください。

本プロジェクトへの参加により、この行動規範に同意したものとみなされます。
