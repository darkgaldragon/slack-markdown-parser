# テスト2: エスケープされたパイプ `\|`

セル内にパイプ文字を含むテーブルです。

| コマンド | 説明 |
|---|---|
| echo "a\|b" | パイプをエスケープして出力 |
| A \| B | OR条件の表記 |
| 通常セル | 問題なし |

確認ポイント: `\|` のバックスラッシュがセルにそのまま表示されるか、それとも除去されて `|` だけになるか。

# テスト3: 結合絵文字（ZWJ）

## 3a. 通常テキスト内の結合絵文字

担当者 👨‍💻 がコードレビュー中です。
家族チーム 👨‍👩‍👧‍👦 で対応しています。
女性エンジニア 👩‍💻 が修正を担当。
虹の旗 🏳️‍🌈 を掲げるイベント。

## 3b. テーブルセル内の結合絵文字

| チーム | 担当 | ステータス |
|---|---|---|
| Backend | 👨‍💻 田中 | **対応中** |
| Frontend | 👩‍💻 鈴木 | *レビュー中* |
| QA | 👨‍👩‍👧 チーム | ~~保留~~ |

## 3c. 装飾記号と結合絵文字の共存

**👨‍💻エンジニアチーム**が修正を完了しました。
*👩‍💻鈴木さん*がレビュー担当です。
`👨‍👩‍👧‍👦`はチームの絵文字です。

確認ポイント: 結合絵文字が分解されず正しく表示されるか（例: 👨‍💻 が 👨💻 に分離しないか）。
