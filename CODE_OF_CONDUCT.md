# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standards

Examples of behavior that contributes to a positive environment:
- respectful communication
- constructive feedback
- empathy for contributors with different backgrounds

Examples of unacceptable behavior:
- harassment or discrimination
- personal attacks
- publishing private information without consent

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that are not aligned with this Code of Conduct.

## Reporting

Please report conduct issues privately to repository maintainers.

By participating, you agree to follow this Code of Conduct.
